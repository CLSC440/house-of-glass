# دليل الربط التقني (Gallery <-> HG System)

يا بشمهندس، الملف ده فيه كل التفاصيل اللي محتاجها عشان تربط الـ **Gallery Website** مع الـ **HG System (DC)**.

الربط دلوقتي شغال بمنطق **Vercel-first**، ومعتمد على إن **Code المنتج** هو اللغة المشتركة بين السيستمين.

---

## 1. إزاي تاخد الصور من عندنا؟

إحنا عاملين API Endpoint على Vercel تقدر تنده عليه بـ Product Code وهو يرجعلك الصور وبيانات المنتج.

### الرابط
```http
GET https://your-vercel-domain/api/media?code=PRODUCT_CODE
```

### شكل الـ Response
```json
{
  "code": "CUP-101",
  "name": "طقم كاسات كريستال",
  "images": [
    "https://ik.imagekit.io/.../image1.jpg",
    "https://ik.imagekit.io/.../image2.jpg"
  ],
  "variants": [
    {
      "name": "أحمر",
      "images": ["https://ik.imagekit.io/.../red-cup.jpg"]
    }
  ]
}
```

ممكن تلاقي بعض المنتجات القديمة راجعة بروابط من مزود صور قديم، فالأفضل تعتبر أي رابط `https` صالح إنه صورة قابلة للعرض.

---

## 2. إزاي تبعت لنا الأسعار والمخزون؟

إحنا محتاجين API عندك في الـ DC يرجع الأسعار والمخزون باستخدام أكواد المنتجات.

الواجهة عندنا بقت تعدي على Vercel proxy الأول بدل ما تكلم رابط الـ DC مباشرة من المتصفح.

### Request مقترح
```json
{
  "codes": ["CUP-101", "PLATE-505", "VASE-99"]
}
```

### Response مقترح
```json
{
  "CUP-101": { "price": 1500, "stock": 5 },
  "PLATE-505": { "price": 400, "stock": 20 },
  "VASE-99": { "price": 0, "stock": 0 }
}
```

السبب إننا بنفضل Bulk request بدل requests كتير منفصلة.

---

## 3. إزاي بنبعت الفواتير للـ DC؟

إرسال الفواتير من عندنا للـ DC بيتم من خلال:

- `api/integrations/online-invoices.js`

وده بيعتمد على متغيرات البيئة دي على Vercel:

- `DC_ONLINE_INVOICE_URL`
- `DC_ONLINE_INVOICE_API_KEY`

يعني مهم جدًا إن بيانات الربط دي تفضل في إعدادات Vercel، مش متحطوطة في الكود.

---

## 4. ملاحظات مهمة

### أ. Firebase Access
لو حبيت تقرأ المنتجات مباشرة من Firebase، لازم يكون عندك `FIREBASE_SERVICE_ACCOUNT` أو ملف service account آمن خارج الكود.

### ب. توحيد الأكواد
حقل الـ `code` عندنا إجباري ومميز لكل منتج.

لازم الأكواد في الـ DC تبقى مطابقة لنفس الأكواد في الجاليري، لأن دي نقطة الربط الأساسية بين الصور والأسعار والمخزون والفواتير.

### ج. متغيرات البيئة على Vercel
الحد الأدنى المطلوب في إعدادات Vercel:

- `FIREBASE_SERVICE_ACCOUNT`
- `DC_ONLINE_INVOICE_URL`
- `DC_ONLINE_INVOICE_API_KEY`
- `DC_PUBLIC_PRODUCTS_URL`
- `DC_PUBLIC_STOCK_URL`
- `IMAGEKIT_PRIVATE_KEY`
- `IMAGEKIT_PUBLIC_KEY`
- `IMAGEKIT_URL_ENDPOINT`
